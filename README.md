## How to use
><h3 id="running-server">
>   🖥️ Running backend (server).
></h3>
>
>> ```bash
>># Access back-end folder.
>> $ cd backend
>>
>># Install dependencies.
>>$ yarn install
>>#or.
>>$ npm install
>>
>># Start the API server.
>>$ yarn dev
>>#or.
>>$ npm dev
>>
>># The back-end server will listen on port 3333.
>># Make API calls on http://localhost:3333.
>>```
>
